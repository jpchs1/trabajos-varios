## 6. Instrucciones para convertir el SVG a JPG/PNG

Tu flyer está en `flyer-menu-instagram.svg`. Para usarlo en Instagram/WhatsApp/Facebook:

**Opción A — Lo más fácil (Canva):**
1. Entra a canva.com (gratis).
2. Crea un diseño "Publicación de Instagram" 1080x1080.
3. Sube el SVG (Subir archivos → SVG).
4. Descarga como PNG o JPG.

**Opción B — Online directo (sin cuenta):**
1. Entra a `cloudconvert.com` o `svgtopng.com`.
2. Sube el SVG.
3. Descarga el PNG.

**Opción C — Desde tu computador (rápido):**
1. Abre `flyer-menu-instagram.svg` con doble clic en cualquier navegador (Chrome, Edge).
2. Toma un screenshot del cuadrado completo.
3. Lo recortas y listo.

**Opción D — Con tu celular:**
1. Manda el SVG a tu WhatsApp.
2. Ábrelo desde el celular (te lo abre el navegador).
3. Captura de pantalla.
